    ########## citation ##########

    The Vienna Ab initio Simulation Package (VASP) is a computer program for atomic scale materials modelling. VASP users are widely distributed, such as chemistry, physics, material, enviornment. Users have developed and contributed scripts or applets used in their field based on VASP. VASPKIT is a Pre- and Post-Processing Program for the VASP code, written by Fortran and used under the Linux environment.

    1. Quick Start

    (1) In order to ensure the full functionality of the VASPKIT, users need to configure the environment of the vaspkit from ~/.vaspkit file. Copy the environment variable file to the user directory by running the following command under the bash terminal.

    cp -f how_to_set_environment_variable ~/.vaspkit

    Edit the .vaspkit file,

    vi ~/.vaspkit

    The configuration file is mainly used to set the environment variables of the vaspkit, including the VASP version information, the directory of the target library, the selection of the functional, and whether to select the POTCAR according to the VASP official recommendation, and so on. 

    # cp how_to_set_environment_variable ~/.vaspkit and modify the ~/.vaspkit 

VASP5                    .TRUE.             # .TRUE. or .FALSE.; Set .FALSE. if vasp.4.x 
GGA_PATH                 '~/POTCAR/GGA'     #  Path of GGA potential. 
PBE_PATH                 '~/POTCAR/PBE'     #  Path of PBE potential. 
LDA_PATH                 '~/POTCAR/LDA'     #  Path of LDA potential. 
POTCAR_TYPE               PBE               #  PBE, GGA or LDA; 
GW_POTCAR                .FALSE.            # .TRUE. or .FALSE.;
RECOMMENDED_POTCAR       .TRUE.             # .TRUE. or .FALSE.; 
MINI_INCAR               .FALSE.            # .TRUE. or .FALSE.; 
SET_FERMI_ENERGY_ZERO    .TRUE.             # .TRUE. or .FALSE.;  
SET_INCAR_WRITE_MODE     BACK-UP-OLD        #  OVERRIDE, APPEND,BACK-UP-OLD,BACK-UP-NEW;

    (2) Setting up local repository pseudopotential directory. For example, unzip the VASP PAW_PBE pseudopotentials at ‘~/POTCAR/PBE’.

    (3) Add the absolute path of the vaspkit to the environment variable. 

    echo 'export PATH=/home/vaspkit.0.80/bin/:$PATH' >> ~/.bashrc
source ~/.bashrc

    Replace the absolute path of the /home/vaspkit.0.73/bin/ directory by user’s own vaspkit executable.

    Vaspkit provides an automatic configuration script setup.sh since version 0.73. Execute

    ​ source setup.sh

    or bash setup.sh

    If ~/.vaspkit already exists, the above operation will not overwrite it and still use the old version of the environment variable.

    2. Generate Input Files

    In order to successfully run VASP tasks, usually one need 4 files: INCAR, POSCAR, POTCAR and KPOINTS.

    1. INCAR file is a control file that tells VASP what tasks to calculate and how to calculate;

    2. POSCAR is a file containing lattice information, atomic coordinate information, and atomic velocity information (for MD);

    3. POTCAR is a pseudopotential file, which is USPP or PAW type;
    4. KPOINTS, which can be included in INCAR, but not recommended for omitting, contains K-points information in the reciprocal space at which the wave function integrates to obtain the charge density.

    2.3 Generate POTCAR

    When generating KPOINTS, POTCAR will also be generate automatically. Or use 103 to generate POTCAR.

    103
  -->> (01) Reading Structural Parameters from POSCAR File...
  -->> (02) Written POTCAR File with the Standard Potential!

    It extracts the elements information from POSCAR and combines the corresponding POTCAR from the pseudopotential folders that you set in ~/.vaspkit.

    GGA_PATH                 '~/POTCAR/GGA'     #  Path of GGA potential. 
PBE_PATH                 '~/POTCAR/PBE'     #  Path of PBE potential. 
LDA_PATH                 '~/POTCAR/LDA'     #  Path of LDA potential.
POTCAR_TYPE               PBE               #  PBE, GGA or LDA; 
RECOMMENDED_POTCAR       .TRUE.             # .TRUE. or .FALSE.;

    Set POTCAR_TYPE to PBE, GGA, orLDA as your requirement. The RECOMMENDED_POTCAR tag control whether to use recommended potentials from VASP manual (Page. 195, 2018.10.29, http://cms.mpi.univie.ac.at/vasp/vasp/Recommended_PAW_potentials_DFT_calculations_using_vasp_5_2.html). If RECOMMENDED_POTCAR is .FALSE., POTCAR with no extensions will be used. If RECOMMENDED_POTCAR is .TRUE., official recommended POTCAR will be used.

    POTCAR types:

    1. No extensions “_”
    2. _d. An extension d, treat the d semi core states as valence state.
    3. _pv or _sv. The extensions _pv and _sv imply that the p and s semi-core states are
      treated as valence states.
    4. _h and _s. An extension _h or_s implies that the potential is harder or softer than the standard potential and hence requires a higher or lower energy cutoff.
    5. Pseudo hydrogen. ex: H.5

    6. _GW. Used for GW calculation.

      If one want to generate POTCAR for GW calculation, Set the GW_POTCAR to .TRUE. in ~/.vaspkit. 

    GW_POTCAR                .FALSE.            # .TRUE. or .FALSE.;

    VASPKIT also provide 104 tag to generate POTCAR manually by selecting the type of potential for each element.

    104
 -->> (1) Reading Structural Parameters from POSCAR File...

    3 Band Structure

    VASPKIT is very powerful at pre- and post- process VASP band structure calculation.

    3.1 Pre-process Band Structure (pure functional)

    irreducible Brillouin zone is the first Brillouin zone reduced by all of the symmetries in the points group of the lattice (point group of the crystal). To do band structure calculation, one should select high symmetry points, and link them along edges of Brillouin zone.

    For instance, FCC metel：

    FCC metel：

    The rule to select K-path is unique. Usually, It is unnecessary to select all the edges of irreducible Brillouin zone. The representative and important line are selected and written as line-mode in KPOINTS file. But for large-scale and high-throughout calculation, there should be a rule to define the path from there symmetry and structural information。pymatgen and seeK-path provide some solutions but only can be used for 3D system. VASPKIT provide a tool to generate K-path for 1D, 2D, and 3D materials based on a systematic rule:

    Y. Hinuma, G. Pizzi, Y. Kumagai, F. Oba, I. Tanaka, Band structure diagram paths based on crystallography, Comp. Mat. Sci. 128, 140 (2017). DOI: 10.1016/j.commatsci.2016.10.015 .

    there is Brillouin zones for 2D materials:

    Other ways to get K-path automatically

    pymatgen（https://pymatgen.org/）Computational Materials Science 49 (2010) 299–312.
    seeK-path（https://www.materialscloud.org/work/tools/seekpath）Computational Materials Science 128 (2017) 140–184.

    VASPKIT use the same method as seeK-path website, and can generate line-mode KPOINTS file automatically off-line.
